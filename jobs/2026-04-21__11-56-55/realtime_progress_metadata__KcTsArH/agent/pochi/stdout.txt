## Task: `progressTask-realtime_progress_metadata__KcTsArH`

**File:** `trigger/progress.ts`

The task:
1. Loops over **10 items**
2. For each item, **awaits `wait.for({ seconds: 1 })`** to simulate work
3. Calculates completion percentage: `Math.round((i / totalItems) * 100)` → `10, 20, ..., 100`
4. Calls **`metadata.set("progress", percentage)`** to update the current progress value (overwritten each iteration)
5. Calls **`metadata.append("logs", message)`** to accumulate a log entry per item
6. Returns a completion summary object when done

These metadata updates are observable in real-time by any subscriber watching the run via Trigger.dev's realtime API.
