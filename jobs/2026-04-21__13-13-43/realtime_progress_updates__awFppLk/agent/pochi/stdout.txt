## Key Features

**Tasks Created:**
1. `progress-task` - Generic task accepting `trialId` parameter
2. `progress-task-trial-1` - Pre-configured for trial-1
3. `progress-task-trial-2` - Pre-configured for trial-2

**Progress Implementation:**
- 5-step workflow with 1-second simulation per step
- Progress updates: 20% → 40% → 60% → 80% → 100%
- Metadata updates using `ctx.run.updateMetadata({ progress: currentProgress })`
- Additional metadata includes: step number, total steps, and trial ID

## Usage

Install dependencies:
```bash
cd /home/user/progress-app
npm install
```

Trigger the task:
```typescript
import { progressTask } from "./trigger/progress";

await progressTask.trigger({
  trialId: "my-custom-trial"
});
```

View real-time progress on the Trigger.dev dashboard or via the Realtime API.
