## Implementation Details

### `refine-text` Task
- **ID**: `refine-text`
- **Purpose**: Takes a string and improves it through text refinement
- **Input**: `text` (string), `trialId` (optional string), `iteration` (optional number)
- **Output**: `refinedText`, `iteration`, `needsFurtherRefinement`
- **Behavior**: Mocks text refinement with iteration-specific improvements

### `ai.tool` Wrapper
```typescript
const refineTool = ai.tool(refineTask);
```

### `agent-loop` Task
- **ID**: `agent-loop`
- **Purpose**: Uses the refine tool in a loop to refine text at least twice
- **Input**: `text` (string), `trialId` (optional string), `maxIterations` (number, default: 3)
- **Output**: `finalText`, `iterations`, `refinementHistory`
- **Behavior**: Runs refinement loop with minimum 2 iterations, tracks history
