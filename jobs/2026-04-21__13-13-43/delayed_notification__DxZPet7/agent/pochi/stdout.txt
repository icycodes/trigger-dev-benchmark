### `trigger/notify.ts` — Key Implementation

```typescript
import { task, wait } from "@trigger.dev/sdk/v3";

const trial_id = "trial-001";

export const delayedNotifyTask = task({
  id: `delayed-notify-${trial_id}`,        // → "delayed-notify-trial-001"
  run: async (payload: { userId: string; message: string }) => {
    const { userId, message } = payload;

    // Pause for 10 seconds without consuming compute resources
    await wait.for({ seconds: 10 });

    return {
      sent: true,
      message,                              // original message returned as-is
    };
  },
});
```

### How it works
- **Task ID**: `delayed-notify-${trial_id}` — interpolated at definition time from the `trial_id` constant.
- **Payload**: Accepts `{ userId: string; message: string }`.
- **`wait.for({ seconds: 10 })`**: Suspends the task for 10 seconds using Trigger.dev's durable wait mechanism — the task checkpoint is released and no compute is consumed during the wait.
- **Return value**: `{ sent: true, message }` — confirms delivery with the original message echoed back.

To run locally, set your `TRIGGER_SECRET_KEY` environment variable and execute `npm run dev`.
